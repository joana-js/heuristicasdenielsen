# 📖 Dossiê Educativo: UX & Design de Interfaces (Trabalho N2)

Este projeto é um dossiê interativo desenvolvido como **Trabalho Acadêmico N2** para a disciplina de UX & Design de Interfaces. O objetivo principal é explicar e ilustrar as **10 Heurísticas de Nielsen** e sua aplicação prática em design de interfaces, além de abordar conceitos fundamentais de Psicologia das Cores e Tipografia.

Através de uma análise comparativa, o site apresenta exemplos reais de interfaces bem e mal projetadas, demonstrando o impacto direto das escolhas de design na experiência emocional e funcional do usuário.

---

## 🎯 O Que Abordamos

- **A Base Teórica:**
  - **Psicologia das Cores:** Como as cores atuam como gatilhos cognitivos e afetam a percepção do usuário.
  - **Tipografia:** A diferença e aplicação de fontes Com Serifa (elegância/autoridade) e Sem Serifa (clareza/eficiência).
- **As 10 Heurísticas de Nielsen:** Um resumo prático dos princípios fundamentais de usabilidade.
- **Estudos de Caso (Análise Comparativa):**
  - 🔴 **Site Ruim:** Auditoria de uma interface frustrante (excesso de informações, falta de controle do usuário, design não minimalista).
  - 🟢 **Site Bom:** Análise de uma experiência fluida (ex: Stripe Checkout), destacando heurísticas bem aplicadas, como consistência e design estético.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias e bibliotecas:

* **[React](https://react.dev/) / [Next.js](https://nextjs.org/):** Framework principal da aplicação.
* **[Tailwind CSS](https://tailwindcss.com/):** Estilização utilitária e responsividade.
* **[Framer Motion](https://motion.dev/):** Animações de interface e transições suaves.
* **[Lucide React](https://lucide.dev/):** Biblioteca de ícones minimalistas.
